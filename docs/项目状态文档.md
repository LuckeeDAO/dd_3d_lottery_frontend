## 🚀 部署选项

### 1. Vercel部署 (推荐)
```bash
# 配置GitHub Secrets后自动部署
git push origin main
```

### 2. Docker部署
```bash
# 构建镜像
docker build -t dd-3d-lottery-frontend .

# 运行容器
docker run -p 3000:80 dd-3d-lottery-frontend
```

### 3. 静态文件部署
```bash
# 构建应用
npm run build

# 部署dist目录到任何静态文件服务器
```
